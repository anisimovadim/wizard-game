# 🎮 Инструкция к игре

## 🕹 Управление персонажем

- **W, A, S, D** — передвижение  
- **Левая кнопка мыши (ЛКМ)** — стрельба магическими шарами  
- **Клавиши 1, 2, 3, 4** — переключение стихии магии  

## ✨ Комбинации стихий

В игре доступны особые комбинации стихий, усиливающие урон:

- **Лёд + Огонь**  
  → Эффект: *термический шок* или *пар*  
  → Урон увеличивается в **2.5 раза**

- **Огонь + Земля**  
  → Эффект: *окаменение*  
  → Урон увеличивается в **2.5 раза**
- **Лёд/Огонь + Воздух**  
    → Эффект: *увеличение длительности наложение эффекта на 5 секунд*
